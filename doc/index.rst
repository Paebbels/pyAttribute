pyAttributes Documentation
##########################

.NET-like Attributes implemented as Python decorators.


Installation/Updates
********************

Installation using PIP
======================

.. code-block:: bash

   pip3 install pyAttributes


Updating using PIP
==================

.. code-block:: bash

   pip3 install -U pyAttributes
